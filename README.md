# swm-calender

## 사용 방법


## 기획 배경
- 멘토링 일정을 팀 간에 효율적으로 공유하여 불필요한 커뮤니케이션 비용을 줄이기 위해 개발된 Extension 용 Api 서버

## 페르소나
- 팀장은 팀을 생성하고 캘린더를 관리
- 팀원은 초대를 받아 합류해서 멘토링 일정을 공유

## 주요 기능
- 팀 생성
- 구글 캘린더와 팀 연결 (자동 생성)
- 팀원 초대 및 팀 합류
- 소마 사이트 -> 구글 캘린더로 멘토링 일정 동기화
- 구글 캘린더 -> 소마 사이트로 멘토링 일정 동기화

## 지원하는 플랫폼
- 구글 캘린더

## 기술 스택
- Kotlin/Spring Boot(4.0.7)
- JDK 21
- MySQL 8.0
- Terraform
- AWS EC2, RDS
- GitHub Actions
